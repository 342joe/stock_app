# 📦 Stock App
Application de gestion de stock développée en **PHP (Architecture MVC)** avec **MySQL**.

## ✅ À propos
Stock App est une application permettant de gérer efficacement :  
- Les produits  
- Les catégories  
- Les utilisateurs  
- Les entrées et sorties de stock  
- Les livraisons  
- Des rapports simples  

Elle est conçue pour fonctionner dans un **supermarché**, une boutique ou un dépôt.

---

## 🚀 Fonctionnalités principales

### 📁 Gestion des catégories
- Création, mise à jour et suppression
- Catégories adaptées aux supermarchés (Épicerie, Boissons, etc.)

### 🛒 Gestion des produits
- Ajout de produits avec image
- Attribution d’une catégorie
- Prix, quantité, date d’ajout
- Recherche et listing

### 🚚 Gestion des livraisons
- Enregistrement des produits livrés
- Suivi des quantités

### 👥 Gestion des utilisateurs
- Rôles : Admin, Stock Manager, Stock Operator, etc.
- CRUD utilisateurs (ajout, suppression, mise à jour)

### 📊 Tableau de bord
- Statistiques rapides
- Produits critiques (faible stock)
